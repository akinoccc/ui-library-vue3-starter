# Blog-ui

## Start

### Install

```shell
pnpm i
```

### Start Dev

```shell
pnpm dev
```

You can import the component to "/example/app.example.vue" which you want to preview.

## Preview

Preview the built components.

```shell
pnpm docs:dev
```

you can see the result **after building the packages**.

## Build

```shell
pnpm build
```
